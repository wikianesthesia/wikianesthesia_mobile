The fluid management calculations is based upon patient demographic information provided in the fields above. Some calculations require additional patient data, for which fields are provided within the calculation. As with all clinical decision support tools, ultimately use clinical judgement when considering these results.
  • Some calculations may have important additional information which can be shown using the question mark icon
